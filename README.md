# AI Fruit Ninja (Computer Vision)

A Python-based game that uses the webcam and Hand Tracking (MediaPipe) to slice fruits in the air.

## Current Status: v0.1 (Skeleton)
- [x] Webcam setup
- [x] Hand detection (MediaPipe)
- [x] Index finger tracking (The Blade)
- [ ] Physics engine (Gravity & Fruit spawning)
- [ ] Scoring system
- [ ] UI & Sound effects

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python main.py`
